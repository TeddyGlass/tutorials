# tutorials
Pythonによるケモインフォマティクスを薬学部の学部生向けに講義したときのまとめを公開しています.  

## このリポジトリで扱うこと
このリポジトリでは, ケモインフォマティクスを目的としたPythonの基本的な使用方法を公開します. 内容は未定ですが, 以下のような内容を想定しています.  
* csvファイルの扱い
* 分子構造の扱い
* 化学構造記述子・フィンガープリントの算出
* 機械学習

## このリポジトリで扱わないこと
このリポジトリでは, Pythonでプログラム開発を行うための環境構築については扱いません.
